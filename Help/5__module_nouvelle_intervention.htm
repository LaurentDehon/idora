5. Module NOUVELLE INTERVENTION

 
 

Nom de dossier

 
Nom de dossier. Le nom qui apparait par défaut est celui du dossier sur lequel l'utilisateur a fait un clic droit pour créer l'intervention dans le module DOSSIER. Si l'intervention est liée à un autre dossier, il est possible de choisir le bon dossier dans cette liste déroulante.
 
Haut

Type d'intervention

 
Type d'intervention. C'est cette donnée qui entre en compte dans les statistiques.
 
Haut

Type de lieu

 
Type de lieu.
 
Haut

Intervention

 
Intervention sur place / différée / à SGS / à REMONDIS.
Si l'intervention est réalisée sur place, les dates et adresses des faits et de l'intervention seront identiques.
Si l'intervention à lieu chez SGS ou REMONDIS, les adresses seront remplies automatiquement.

Code postal de l'intervention

Commune de l'intervention. Un système de filtre est intégré au fur et à mesure que l'utilisateur tappe le nom de la commune recherchée. L'utilisateur doit tapper les premières lettres de la commune.
